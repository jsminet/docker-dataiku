# Get the Oracle / Postgres database drivers 

Launch the above command from this directory using gitbash or whatever

```
curl -o ojdbc8-21.13.0.0.jar https://repo1.maven.org/maven2/com/oracle/database/jdbc/ojdbc8/21.13.0.0/ojdbc8-21.13.0.0.jar
curl -o postgresql-42.7.3.jar https://repo1.maven.org/maven2/org/postgresql/postgresql/42.7.3/postgresql-42.7.3.jar
```
